# Shortest Job First (Non-Preemptive) CPU Scheduling

This project implements the Shortest Job First (SJF) CPU Scheduling Algorithm using C.

## Input Format
n
PID AT BT

Example:
4
P1 0 6
P2 1 8
P3 2 7
P4 3 3

## Output
Waiting Time
Turnaround Time
Average Waiting Time
Average Turnaround Time

## Compile

gcc sjf.c -o sjf

## Run

./sjf
